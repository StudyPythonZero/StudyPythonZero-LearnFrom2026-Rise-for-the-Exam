# ubuntu安装python

## 更新软件包列表

```linux
sudo apt update
```

## 安装python

```linux
sudo apt install python3
```

## 安装python-pip

```linux
sudo apt install python3-pip
```

## 设置python命令（可选）

```linux
sudo apt install python-is-python3
```

## 检查安装情况

- 未设置python命令

```linux
python3 --version
pip3 --version
```

- 已设置python命令（可选）

```linux
python --version
pip --version
```
