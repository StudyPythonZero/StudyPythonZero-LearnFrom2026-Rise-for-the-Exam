## get_string

```c
char* answer = get_string("What's your name?");
```

- 如果想保留函数的返回值，使用等号即可
- 等号左边写上用来存放返回值的变量名
- 记得使用英文输入的分号结束语句

### print variable 打印变量

- 如果我想打印返回的值

    ```c
    printf("hello, answer\n");
    ```

- 会输出

    ```c
    "hello, answer
    ```

- 但目标是打印answer存储的值，需要修改

    ```c
    printf("hello, %s\n", answer);
    ```

- %s 表示一个占位符，用于表示写代码的时候还不知道具体内容的字符串
  - 运行的时候会被实际内容填充，并用实际内容替换占位符
  - 在字符串后面添加英文符号的逗号，并添加需要替换占位符的变量
- 输入 Lihua 结果输出

    ```c
    hello, Lihua
    ```

## 命令行

- rm 删除文件
- mkdir 创建文件夹
- mv 移动文件，第一个指定源文件，第二个指定目标位置
  - 可以用于重命名
  - .. 表示上一级目录

    ```txt
    mv get_string.c 1.2get_string // 移动到文件夹1.2内
    mv hello.c old.c // 文件名称从 hello.c 改成 old.c
    ```

- ls 列出文件夹里的所有文件
- cp 复制文件并对复制文件命名

    ```txt
    cp hello.c old.c // 复制 hello.c 文件，并命名为 old.c
    ```

- cd 移动到指定目录
- rmdir 删除空文件夹

## conditionals 条件语句

```c
if (x < y)
{
    printf("x is less than y\n"); // ...
}
else if (x > y)
{
    printf("x is not less than y\n");
}
else if (x == y)
{
    printf("x = y\n");
}
```

- 花括号表示 if 语句通过后执行的内容
- 缩进一般设定为 4 空格
- 布尔表达式只有两个结果 是 或 否
- 双等号是相等运算符，单个等号是赋值运算符
