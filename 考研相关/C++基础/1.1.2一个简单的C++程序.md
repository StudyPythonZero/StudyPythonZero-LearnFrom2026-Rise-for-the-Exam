# 概述

## 数据处理

- 不论是什么操作，数据处理都可以简化成：输入 -> 计算 -> 输出
- 数据来源/输入通道可以不同
- 处理方式可以不同
- 输出方式可以不同

- 数据描述：存放半径（从外部获取）、面积（运算结果）、周长（运算结果）的实型数，一个固定不变的圆周率PI
- 数据处理：输入 -> 计算 -> 输出
  - c++是控制台程序
  - cout输出文本提示用户输入
  - 用户输入程序通过cin存放到r
  - 数据计算
  - 输出结果显示在屏幕上
- 注释
  - // 表示注释行，可以在任意行设置，辅助他人了解相关信息
  - /**/ 表示字符串注释，内部的字符串全都是注释内容
  - 调试程序的辅助手段，可以通过对程序部分注释观察输出结果
- 预编译指令
  - iostream 标准输入输出流
  - 数据描述、数据处理相关操作
- 标准命名空间
  - 用于防止同名冲突
  - 可以使用外部的预定义功能
- 函数
  - 一个功能模块的封装
  - 函数头格式： 函数返回值类型 函数名称（参数）
  - 主函数：必须也只能有一个，返回值类型一般取 int 或是 void
  - 主函数名称由系统预定义，必须为 main
  - （）内是参数表，可以没有参数，但是括号不能省略
  - 函数体：{}花括号内的语句序列
  - 每个语句以分号;结束
  - 一个语句可以分多行书写
  - 尽量避免一行写多行语句
  - 需要培养良好的书写风格
  - 函数是程序的基本运行单位，一般形式为

  ``` txt
  类型 函数名（参数表）
  {
    语句序列
  }
  ```

## 面向对象程序设计

- 程序设计方法分为两种
  - 面向程序的结构化设计方法
  - 面向对象的程序设计方法

- 圆是一个抽象的类类型
  - 圆有自己的属性，例如半径，颜色等
  - 圆有自己的方法，例如移动自己的坐标、提供自己的周长、面积等

- 一个 类类型 有自己的具体属性就被称为对象
- 一个 类类型 可以做的事情被称为方法
- C++里 类类型 的方法被称为函数，或者是这个对象可以做的 成员函数
- 根据开头的程序，可以设置圆的类类型对象
  - 数据成员：半径
  - 成员函数：置半径值、求圆的半径、求圆的周长、求圆的面积
- 根据以上内容，我们可以编写程序

``` C++
#include<iostream>
using namespace std;

class Circle
{
  // 数据成员
  double radius;
  // 成员函数
  public:
    void Set_Radius(double r){radius = r;}
    double Get_Radius() {return radius;}
    double Get_Girth() {return 2 * 3.14 * radius;}
    double Get_Area() {return 3.14 * radius * radius;}
};

int main()
{
  // 创建实例
  Circle A, B;
  A.Set_Radius(6.23);
  cout << "A.Radius = " << A.Get_Radius() << endl;
  cout << "A.Girth = " << A.Get_Girth() << endl;
  cout << "A.Area = " << A.Get_Area() << endl;
  B.Set_Radius(10.5);
  cout << "B.Radius = " << B.Get_Radius() << endl;
  cout << "B.Girth = " << B.Get_Girth() << endl;
  cout << "B.Area = " << B.Get_Area() << endl;
  return 0;
}
```
