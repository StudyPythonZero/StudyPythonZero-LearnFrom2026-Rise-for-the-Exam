### 1.1 拆解

``` C++
#include<iostream>
/* 引用一个头文件，名称是 输入输出流
 * <iostream>文件里包含的两个知名使用方法 cin 和 cout
 * 必须通过这种方式引用才能使用 <> 内的文件/文件夹包含的方法 */

 using namespace std;
 /* C++使用文件里的方法，除了开始的导入，还需要注明方法属于头文件里面的哪个大类（大概、理解有偏差）
  * 比如原文中的 cout 属于 std 类
  * 如果没有这行，后面的所有 sout 方法都应改为 std::cout
  * 这段话的意思大概是，将所有 没有标注大类的，默认改为 std ，也就是我就算只写了 cout，它也会默认在前面加上 std:: 
  * 自己琢磨的，应该有误，等以后有空重新查询的时候再修改吧（2026.6.28）*/

int main()
/* 主函数，程序最开始跑的地方，从main函数往下读取
 * int 表示会返回一个 int 类型的值
 * 一般搭配 return 0; 使用 */

{
    cout << "hello world!" << endl;
    /* c++ 的 out 方法 << 需要打印的内容 << 换行
     * 也可以写成 cont << "hello world!\n";
     * c++ 语言的 out 方法，这样理解就行
     * cin 同理，c++ 语言的 in 方法
     * cout 在控制台打印内容
     * cin 在控制台获取用户输入的内容
     * << 额，我也不懂，应该是把右边的内容输入到左边，再加一个 << 那就是从右往左拼接，然后将内容填入最左边的方法里面，理解不了一点（2026.6.28）*/

    return 0;
    // 在main函数里面说了，搭配 int 类型函数使用，返回一个0，作为运行无异常，有异常编译期间会报错，返回值就自动不是0了
}
```
