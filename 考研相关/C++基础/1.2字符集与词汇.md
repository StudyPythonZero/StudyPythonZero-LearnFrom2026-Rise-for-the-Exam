# 概述

- 所有的语言系统都是由字符集和规则集组成的
- “字符”是语言的不可分解的最基本语法单位
- 由字符可以组成“词”，由词组成“表达式”、“句子”，又由各种句子构成“函数”、“程序”
- 单词是语言中有意义的最小语法单位。根据形成规则，一个单词由一个或多个字符组成

## 字符集

- C++语言的字符集是ASCII码的子集，其中包括
  - 26个小写字母

  ```txt
  abcdefghijklmnopqrstuvwsyz
  ```

  - 26个大写字母

  ```txt
  ABCDEFGHIJKLMNOPQRSTUVWSYZ
  ```

  - 10个数字

  ```txt
  0123456789
  ```

  - 其他符号，参考C++字符集和编码

## 词汇

- 1.关键字
  - 又称保留字
  - 关键字是系统预定义的单词。C++不允许对关键字重定义。
  - 常用关键字

  ```txt
  asm else  new this
  auto  enum  operator  throw
  bool  explicit  private true
  break export  protected try
  case  extern  public  typedef
  catch false register  typeid
  char  float reinterpret_cast  typename
  class for return  union
  const friend  short unsigned
  const_cast  goto  signed  using
  continue  if  sizeof  virtual
  default inline  static  void
  delete  int static_cast volatile
  do  long  struct  wchar_t
  double  mutable switch  while
  dynamic_cast  namespace template
  ```

- 2.标识符
  - 由程序员定义的命名符
  - 以字母或下划线开始，由字母、数字和下划线组成的字符串
  - 不能使用关键字作为用户标识符
  - C++中，字母大小写敏感，Aa和aa是两个不同的标识符
  - C++没有规定标识符的长度，不同编译系统有不同的识别长度
  - 标识符尽可能做到见文知义
- 3.运算符
  - 运算符是对数据进行操作的简洁表达，以单词的形式调用系统预定义函数
  
  ```c++
  + - * / > < >= <= == !=
  ```

- 4.分隔符
  - 用于在程序分隔不同的语法单位，便于编译系统的识别
  - 常用的分隔符有：空格，逗号，分号，冒号，括号，注释符等
  - 不同性质的单词连接，单词本身也起到分隔符的作用
  - 例如
  
  ```C++
  int a, b, c;
  a=b+c;
  ```

- 5.常数
  - 常数由数据的书写形式定义它的类型和值
  - 基本类型常数在程序运行时直接参与运算，不占用内存存储
  - 例如

  ```c++
  500 // 整数 or 实数
  3.14159 // 浮点数 or 实数
  0.263e-10 // 浮点数 or 实数
  'N' // 字符
  ```
